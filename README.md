# Multi-perspective-debater

This project hosts agents with different personas to provide diverse perspectives on user queries. The agents are designed to simulate various debating styles and approaches.

## Features
- **Good Agent (Optimist):** Highlights strengths and opportunities.
- **Devil Agent:** Challenges ideas and explores potential flaws.

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/<USERNAME>/Multi-perspective-debater.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Set up environment variables:
   - Create a `.env` file and add your API keys.

## Usage
Run the agents individually to interact with their unique personas:
```bash
python good_agent.py
python devil_agent.py
```
